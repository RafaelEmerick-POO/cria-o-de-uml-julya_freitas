# Requisito do Laboratorio

Os dois codigos deste repositorio representam o mesmo dominio de forma simples:

- um `Equipamento` com identificacao, descricao e estado operacional;
- um `SensorTemperatura` especializado a partir do equipamento base;
- um fluxo pequeno em `main` para mostrar o uso das classes.

Use esse contexto apenas para orientar a leitura.
O principal material de analise da atividade e o proprio codigo em `src_cpp/` e `src_python/`.
